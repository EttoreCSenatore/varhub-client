VARhub Offline Mode Helper

Having trouble connecting to the VARhub server?

This page will help you enable offline mode to continue using the application with sample data.

1Open Developer Console

First, you need to open your browser's developer console:

  • Chrome or Edge: Press F12 or Ctrl+Shift+J (Windows/Linux) or Cmd+Option+J (Mac)
  • Firefox: Press F12 or Ctrl+Shift+K (Windows/Linux) or Cmd+Option+K (Mac)
  • Safari: Press Cmd+Option+C and ensure the developer menu is enabled in preferences

2Navigate to Console Tab

Once the developer tools are open, click on the "Console" tab.

3Enter Command to Enable Offline Mode

Copy and paste the following command into the console and press Enter:

localStorage.setItem('useMockData', 'true'); localStorage.setItem('autoOfflineMode', 'true'); alert('Offline mode enabled! The page will now reload.'); window.location.reload();

4Use the Application

After the page reloads, you should see an "OFFLINE" indicator in the navigation bar, and you'll be able to use the application with sample data.

5Return to Online Mode (When Available)

To switch back to online mode when the server is available:

  1. Click the "Go Online" button in the navigation bar, or
  2. Enter the following in the console:
localStorage.removeItem('useMockData'); localStorage.removeItem('autoOfflineMode'); alert('Online mode enabled! The page will now reload.'); window.location.reload();

Click the button above to enable offline mode with a single click.
